## Getting Started (for devs)

First, run the development server:

```bash
npm run dev
```

Then run the inngest server:

```bash
npm run inngest
```

Then run the docker db server:

```bash
sudo docker-compose up
```

(Optional) Run the drizzle studio to see the PostgreSQL database:

```bash
npm run db:studio
```
